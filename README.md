How to run mailpit 
1. cd C:\Users\Ldacanay\mailpit or where u extract 
2. mailpit.exe to run the mailpit

.env setup to make it work:
    MAIL_MAILER=smtp
    MAIL_SCHEME=null
    MAIL_HOST=127.0.0.1
    MAIL_PORT=1025
    MAIL_USERNAME=null
    MAIL_PASSWORD=null
    MAIL_FROM_ADDRESS="hello@example.com"
    MAIL_FROM_NAME="${APP_NAME}"
